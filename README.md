# React-App-OMDb-API
React App OMDb API

## Comandos
- `npm init -y` : Crea archivo de configuracion package.json
## Modulos
- `npm i webpack `
- `npm i webpack-cli`
- `npm i babel-loader`
- `npm i @babel/core`
- `npm i @babel/preset-env`
- `npm i @babel/preset-react`

```bash
npm i webpack webpack-cli babel-loader @babel/core @babel/preset-env @babel/preset-react
```
